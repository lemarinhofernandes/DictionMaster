# Instalation

**Hey you!** Want to get some word definition? <br>
Come and try Diction Master! Here's what you need to do:

```
git clone https://github.com/lemarinhofernandes/DictionMaster
cd DictionMaster/
pod install
open DictionMaster.xcworkspace
```

### FYI
We're using:
- mvvm-c
- delegates
- factory
- repository
- UIKit (viewCode)
- auto layout
- Pods (Alamofire)
- local cache
